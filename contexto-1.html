Área:Comunicación

Sistema auxiliar de comunicación y actividades
diarias para pacientes con Escloriosis Lateral Amiotrófica

Este artículo presenta el desarrollo de un sistema de comunicación para ayudar a pacientes con ELA en una etapa avanzada que pueden mover solo los párpados y a los pacientes en etapas intermedias que pueden mover la cabeza pero no hablar. El sistema fue desarrollado en la tarjeta Raspberry PI 3 (RPI 3) utilizando el software Python que usa bibliotecas para facilitar la interpretación y comunicación de estos pacientes. El sistema incluye una ventana principal que simula un teclado virtual en el que el cursor del ratón se mueve sobre cada letra. En el teclado se pueden encontrar las opciones de borrar, volver atrás, control domótico, escuchar música y acceder a una subventana donde se encuentran los mensajes predefinidos que pueden ser rápidamente identificados y seleccionados por el paciente. El mouse se desplaza por cada opción por un tiempo determinado y si el cursor está en la opción deseada se selecciona al mover el párpado Para un paciente avanzado, el algoritmo desarrollado en Python detecta la señal enviada a la tarjeta RPI 3 por el sensor de proximidad TCRT5000 cuando se realiza el movimiento voluntario del párpado y simula el clic izquierdo del ratón para controlar el sistema. Para pacientes con ELA en etapa intermedia, se agregó una cámara web conectada al RPI 3 como medio de comunicación alternativo, ya que pueden controlar el mouse moviendo la cabeza.
