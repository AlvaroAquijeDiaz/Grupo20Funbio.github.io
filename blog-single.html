Creación de la página web

  • Semana 1

Para iniciar con la creación de la página web utilizamos un correo grupal con el que nos inscribimos en la plataforma de GitHub. GitHub nos proporciona la opción de crear repositorios para el proyecto.

----

Repositorio

Agregamos un repositorio llamado "Grupo20Funbio.github.io" en el cual creamos nuestra página.Posteriormente, se modificó la configuración para que la página sea pública y pueda ser visualizada

Finalmente, agregamos carpetas con cada una de las secciones requeridas e iniciamos con el diseño de la página
