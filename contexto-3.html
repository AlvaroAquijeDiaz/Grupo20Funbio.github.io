Área:Comunicación

Sistema de comunicación basado en código Morse
en pacientes con esclerosis lateral amiotrófica

Esta aplicación se basa en técnicas de reconocimiento facial y rasgos faciales para extraer Regiones de interés (ROI) mediante la aplicación de filtros Haar y clasificador en cascada con OpenCV implementado en Visual Studio C #. Asimismo, se desarrolló un programa informático CodeBlink para transformar parpadeos en palabras, junto con una interfaz gráfica de usuario, en la que se muestran las letras del alfabeto en español y frases predefinidas., Como consecuencia, el paciente con ELA tiene la posibilidad de estructurar oraciones, así, facilitando su proceso de comunicación con familiares y personal médico. El algoritmo consta de cinco partes: adquisición, procesamiento,codificación, interpretación y visualización. Durante el proceso de adquisición, el usuario identifica y elige una cámara conectada al ordenador. Luego de elegir la cámara, el sistema comienza a capturar imágenes para proceder con la fase de procesamiento, donde se aplican filtros a esas imágenes con el fin de obtener imágenes en escala de grises. Además, en esta etapa se detectan ojos y rostros. En la fase de codificación, cada parpadeo se captura considerando su duración, para decidir si es largo o corto. Finalmente, se construye la palabra codificada, que tendrá que ser interpretada utilizando código Morse. En la etapa de visualización cada palabra se muestra como un mensaje del paciente.
